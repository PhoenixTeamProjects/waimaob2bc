# Phoenix 个人网站后台内容模型

本模型来自通用外贸网站规范的“后台可维护”思想，但已经改造为个人品牌博客站。

## 必备模块

| 模块 | 用途 | 备注 |
| --- | --- | --- |
| posts | 文章、教程、复盘、观点 | 支持 SEO、封面、阅读时间、关联内容 |
| categories | 栏目 | 不作为产品分类使用 |
| topics | 专题 | 外贸运营、社媒运营、AI 实战、AI 建站等 |
| tags | 标签 | 工具、平台、方法、项目阶段 |
| authors | 作者 | 默认 Phoenix，可扩展团队作者 |
| projects | 项目/案例 | 记录真实过程，可标记进行中 |
| courses | 课程/培训 | 不写虚假价格和承诺 |
| media_coverage | 媒体资料 | 只使用真实可核验资料 |
| trust_metrics | 信任数据 | 只使用已确认数据 |
| contact_submissions | 咨询提交 | 存储表单线索 |
| navigation | 导航 | 页眉以后续用户确认为准 |
| site_settings | 网站设置 | Logo、头像、SEO、社媒、统计等 |

## 禁止

- 禁止产品中心、产品分类、产品详情系统。
- 禁止写入用户已声明需要隐藏的私人业务信息。
- 禁止把所有内容塞入一个万能内容表。
