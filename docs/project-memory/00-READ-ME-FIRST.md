# 00 - Read Me First

本目录是 Phoenix 个人外贸网站的项目记忆库。任何 Agent 开始工作前必须先读本文件和 `PROJECT-MEMORY.md`。

## 当前不可变边界

- 网站是中文为主的 Phoenix 个人品牌、实战博客、课程与咨询网站。
- 网站不是产品型 B2B 企业站。
- 不建设产品中心、产品分类、产品详情页。
- 页眉菜单等待 Phoenix 后续提供，不照搬外贸 B2B 通用规范。
- 通用规范只吸收工程框架、部署框架、后台可维护思想、安全红线和交接规范。

## 部署模式

- GitHub：源码主线。
- Hostinger：从 GitHub 拉代码，按根目录 `package.json` 构建前台并发布。
- VPS：后台、数据库、备份、日志、反代或辅助服务框架。
